# Onboarding

New users begin in a limited onboarding flow before approval.

## Flow

1. Select language.
2. Select country.
3. Confirm suggested timezone.
4. Choose 12h or 24h time format.
5. Read role-specific introduction.
6. Wait for approval.

Pending users cannot access operational screens.

## Supported Languages

- English
- Spanish
- Portuguese
- Tagalog / Filipino
- Serbian

## Country And Timezone Quick Picks

- United States: `America/New_York`, `America/Chicago`, `America/Los_Angeles`
- Philippines: `Asia/Manila`
- Serbia: `Europe/Belgrade`
- Colombia: `America/Bogota`
- Brazil: `America/Sao_Paulo`
- United Kingdom: `Europe/London`

## Manager Follow-Up

After the user is approved, managers should confirm role, timezone, availability, Help Center viewing, and onboarding completion in Team QA.
