# Opportunity Command Center

The Opportunity Command Center turns the manual opportunity foundation into a daily operating surface for managers and chatters.

## Purpose

When a user opens Opportunities, they should quickly see:

- New opportunities.
- Reviewing opportunities.
- Assigned opportunities.
- Completed opportunities.
- Rejected opportunities.
- High-priority opportunities.
- Recent human-recorded results.

## Telegram Screens

Opportunities -> Command Center shows:

- Queue counts.
- Top opportunities by score.
- High-priority opportunities.
- Recent results.

Managers also have Opportunities -> Manager View with:

- Team opportunities.
- Unassigned opportunities.
- High-priority opportunities.
- Top performing angles.
- Most active chatters.

Chatters use Chatter Workspace for:

- today's opportunities.
- assigned opportunities.
- assigned models.
- assigned tasks.
- recent results.
- recommended next action.

## Comment Strategy

Opportunity Detail includes Suggested Strategies.

Strategies are deterministic prompts for human review only:

- curiosity.
- question.
- agreement.
- story.
- authority.
- contrarian.
- educational.

They include curiosity, engagement, and risk scores so staff can decide carefully. Agency OS does not post them automatically.

## Results

Opportunity results support:

- `posted`.
- `skipped`.
- `failed`.
- `rejected`.

These results feed Learning Events, Outcome Memory, and opportunity confidence adjustments.
