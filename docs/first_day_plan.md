# First Day Plan

The First Day Plan is a short owner/manager checklist for turning an empty Agency OS into a usable operating system.

Telegram path:

`Owner Home -> First Day Plan`

## Checklist

1. Create first model.
2. Add IG/X/OF accounts.
3. Assign manager.
4. Assign chatters/VAs.
5. Add top creators to watch.
6. Create first opportunities.
7. Assign opportunities.
8. Generate daily briefing.
9. Review team activation.
10. Check production status.

## Progress

Progress is stored in `first_day_checklists` and can be auto-updated from real database state where safe.

The checklist is not a hard gate. It is a calm operator guide so the owner knows what still needs attention.

## Recommended Use

Before adding a full team:

- create at least one real model/brand;
- attach the model's key accounts;
- assign at least one manager or chatter;
- create one test opportunity;
- verify the bot still shows production status as healthy.
